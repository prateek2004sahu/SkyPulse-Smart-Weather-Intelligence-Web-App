# 🌦 SkyPulse – Smart Weather Intelligence

SkyPulse is a modern weather web application that provides real-time weather information, smart analysis, and predictions for any city in the world. The app uses the OpenWeather API and presents data in a clean, animated dashboard with interactive features.

This project demonstrates front-end web development using HTML, CSS, and JavaScript along with API integration.

---

## 🚀 Features

• 🌍 Search weather by city name  
• 🌡 Real-time temperature and weather condition  
• 💧 Humidity and atmospheric pressure information  
• 🌬 Wind speed and direction display  
• 📊 Smart weather analysis summary  
• 🔮 Basic weather prediction based on conditions  
• 🗺 Embedded Google Maps location view  
• 📄 Downloadable weather report in PDF format  
• 🌙 Dynamic UI with modern glassmorphism design  
• 🎨 Animated gradient background and hover effects  

---

## 🛠 Technologies Used

- **HTML5** – Structure of the web app  
- **CSS3** – Styling and animations  
- **JavaScript (ES6)** – Logic and API integration  
- **OpenWeather API** – Weather data source  
- **jsPDF** – Generate downloadable weather reports  
- **Google Maps Embed** – City location visualization
